# Latinščina

Abeceda: A B C D E F G H I K L M N O P Q R S T U V X Y Z

C = /k/; C+i/e = /c/
TI+sam. = /ci/
AE = /e/
OE = /ö/ /e/

## Besedoslovje

### Sklanjatve

L = ločilnik (*ablativ*)
Z = zvalnik (*vokativ*)

### Predlogi

"kam" – tožilnik (*akuzativ*)
"kje" – ločilnik (*ablativ*)
"s/z" (*cum*) – ločilnik (če ni v pomenu orodnika)
