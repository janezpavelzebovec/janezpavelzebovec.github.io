# janezpavelzebovec.net

To je moja osebna spletna stran, kjer hranim znanje o tem, kar me zanima ali je potrebno vedeti.
Usmerjena je predvsem v znanost in [jezikoslovje](https://janezpavelzebovec.net/jezik), vključenih pa je nekaj mojih ljubših ali meni zanimivejših pisateljev, oz. njihovih del; ter [rodoslovna stran](https://janezpavelzebovec.net/rodoslovje).

Stran je še v zgodnjem nastajanju, zato več stvari žal še ne deluje.

Največji, oz. najzahtevnejši, a hkrati tudi najzanimivejši je projekt [Zvezdoslovne (astronomske) ure](https://janezpavelzebovec.net/zvezdoslovna_ura). Za zahtevnejše SVG-je je uporabljen tudi Python. Za ralago posameznih delov in branja Ure glej tudi [Opombe k zvezdoslovni uri](https://janezpavelzebovec.net/opombe_k_zvezdoslovni_uri).
